Universal Basic Income

How it can solve wealth inequality

What Is Universal Basic Income?

Universal Basic Income is a program where a government provides a set amount of money to every citizen or resident on a regular basis.

What Can It Be Used For?

Universal Basic Income is a cash transfer program, so it can be used for anything, including:
  • Housing
  • Education
  • Medical Bills
  • Food
  • Utilities
  • Vacations

What is Wealth Inequality?

The main problem Universal Basic Income is intended to solve is wealth inequality. Wealth inequality is how wealth, defined as money and assets, is unevenly spread across a population. Wealth inequality has grown dramatically since 1990, and currently the top 10% of income earners own over 68% of wealth.

Why does Wealth Inequality exit?

Wealth builds up in higher percentiles for different economic reasons:
Systemic:
  • Automation
  • Outsourcing
  • Tax Systems
  • Financial Deregulation
Individual:
  • Education Opportunities
  • Inheritance

What Should be known about it?

Universal Basic Income can help in many ways, but there can also be some drawbacks.
Cons:
  • Expensive
  • Can Cause Inflation
  • Can depress work participation
Pros:
  • Provides financial security
  • Reduces financial stress
  • Transfers money to everyone
  • Increases economic demand
